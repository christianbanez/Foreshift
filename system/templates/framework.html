Algorithm

Multivariate Adaptive Regression Splines (MARS) Algorithm

Multivariate Adaptive Regression Splines (MARS) is a decision tree algorithm used for non-linear regression problems. It is a supervised machine learning that requires data labeling amongst the gathered data for the model, this is to provide context for the machine learning model to learn from the data. The algorithm was developed by Jerome H. Friedman as a new approach to flexible regression modelling of significant data. According to Friedman (1991), “this method produces continuous models with continuous derivatives."

Furthermore, MARS algorithm mainly uses piecewise linear basis functions – resulting for the regression surface to build up parsimoniously. Multivariate adaptive regression splines (MARS) algorithm is one of the few tree-based data mining algorithms in predicting.
